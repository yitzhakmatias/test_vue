# vue-test-parking

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
First run services and after serve in cmd
```
yan services
yarn serve
```

The app is running on http://localhost:8080/


### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
